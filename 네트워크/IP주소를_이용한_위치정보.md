# 2.4.4

- 위치 정보를 얻는 방법
    - GPS를 통해 위도 및 경도 값을 구하기
    - 연결된 Network 정보를 통해 현위치를 짐작

- IP 주소는 인터넷에서 사용하는 네트워크 주소
    - IP 주소를 통해 동 or 구까지의 위치 추적 가능
    - IP 주소를 기반으로 위치를 찾는 사이트
        - https://mylocation.co.kr/
        - https://iplogger.org/my-ip/
